## Installation (Arch):
### From Arch official repositories:
```
sudo pacman -S
```
- git
- i3
- picom
- xrandr
- nitrogen
- rofi
- dunst
- polybar
- yay
- zsh (w/ oh-my-zsh and powerlevel10k theme)
### From the AUR:
Install `yay` first
```
sudo pacman -S --needed git base-devel && git clone https://aur.archlinux.org/yay.git && cd yay && makepkg -si
```
then run:
```
yay -S
```
- nerd-fonts-complete
